unambiguous_year(year)

Extract a 4-digit year from a 2-digit year. If the number is less than 20, assume year 20#{number} otherwise use 19#{number}. Ignore if already 4 digits.

Eg: 

   10 => 2010, 60 => 1960, 00 => 2000, 1963 => 1963

[Validate]
